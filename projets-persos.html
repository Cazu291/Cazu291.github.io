La page github de mon simsimi

Mon premier projet a été, quand j'ai découvert l'édition de fichiers depuis python, de recréer une copie de simsimi (une application téléphone qui avait étée créée pour apprendre de ce que lui disent les gens, elle avait fini par être grossière, on se demande pourquoi...). Cette version est primitive, elle reconnaît une syntaxe exacte et non une approchée, comme le faisait simsimi.
